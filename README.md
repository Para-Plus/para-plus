# Para-Plus

Marketplace tunisien de parapharmacie et produits médicaux.

## Technologies

### Backend
- Django 4.2.9
- Django REST Framework
- MongoDB Atlas avec MongoEngine
- Google OAuth 2.0
- JWT Authentication
- Cloudinary (stockage images)
- Déployé sur Render

### Frontend
- Next.js
- TypeScript
- TailwindCSS
- Déployé sur Vercel

## URLs

- **Frontend**: https://para-plus-tn.vercel.app
- **Backend API**: https://para-plus-tn.onrender.com/api
- **Health Check**: https://para-plus-tn.onrender.com/health
